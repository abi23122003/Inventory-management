# Swoosh World

A production-ready full-stack React app template (Fusion Starter) with Vite + React 18 + TailwindCSS, Express API, TypeScript, Vitest, and Radix UI.

## Prerequisites
- Node.js LTS
- pnpm (`npm install -g pnpm`)

## Getting Started
```powershell
pnpm install
pnpm dev
```
Dev server runs client + server with hot reload.

## Scripts
```powershell
pnpm build      # Production build
pnpm start      # Start production server
pnpm typecheck  # TypeScript validation
pnpm test       # Run Vitest tests
```

## Deploy
- Netlify: uses `netlify.toml` and `netlify/functions`
- Vercel: supported (configure build and output)
